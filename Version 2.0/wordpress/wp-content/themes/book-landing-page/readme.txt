== Description ==

Book Landing Page is a beautiful WordPress theme suitable for creating landing page for your book. The home page is thoughtfully designed to increase leads and purchase of the book. The theme is easy to use and doesn’t require coding experienc. You can launch your book in less than 10 minutes with Book Landing Page and market your book. Book Landing Page is mobile friendly. It is also translation ready. Check the demo at https://demo.rarathemes.com/book-landing-page/, documentation at https://docs.rarathemes.com/docs/book-landing-page/, and get support at https://rarathemes.com/support-ticket/.

== Installation ==
        
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Documentation For Homepage Setup

A. Banner Section:

** Important Note **
  * The Featured Image of  the Page ( with Homepage Template) selected as  Front page will appear as a Banner Image if banner is enabled.
  * The Title and content of the page of the Homepage will appear on the banner

  1. Go to Appearance> Customize> Homepage Settings>Banner Section
  2. Check to Enable Banner Section.
  3. Enter Banner Section button Text and Banner section button URL 
  4. Choose Image for the accepted card Image
  5. Click Save and Publish. 

B. Features Section 

  1. Go to Appearance> Customize>Homepage Settings>Features Section
  2. Check Enable Features Section to display it on the homepage
  3. Enter Features Section Title and Features Section Description 
  4. Select Post One, Post Two, Post Three, Post Four, Post Five and Post Six   for Features Section.
  5. Click Save and Publish. 

C. Testimonial Section:

  1. Go to Appearance> Customize>Homepage Settings>Testimonial Section
  2. Check Enable Testimonial Section to display it on the homepage
  3. Enter Testimonial Section  Title and Testimonial Section Description 
  4. Select Post One and Post Two. 
  5. Click Save & Publish 

D. Review Section:

  1. Go to Appearance> Customize>Homepage Settings>Review Section
  2. Check Enable Review Section to display it on the homepage
  3. Enter Review Section  Title and Review Section Description.
  4. Enter  Review Video Embed Link. ( If the video is not select the featured Image of the post will appear on the section.
  5. Select a post for the section
  6. Click Save & Publish

E. Sample Menu Settings

  1. Go to Appearance> Customize>Homepage Settings>Sample Menu Settings
  2. Check Enable Sample Menu Section to display it on the homepage
  3. Enter Sample Menu Section  Title and Sample Menu  Section Description.
  4. Select Post One, Post two , Post Three, Post Four, Post Five.
  5. Click Save & Publish 

F. About Section:

  1. Go to Appearance> Customize>Homepage Settings>About Section
  2. Check Enable About Section to display it on the homepage
  3. Enter About Section  Title and About Section Description.
  4. Enter About Video Embed Link. ( If the video is not select the featured Image of the post will appear on the section.
  5. Select a post for the section
  6. Click Save & Publish

G. Promotional Section

  1. Go to Appearance> Customize>Promotional Section
  2. Check Enable Promotional Section to display it on the homepage
  3. Select post for the section.
  4. Enter Promotional  Section  button Text and Promotional  Section Button Link. 
  5. Upload accepted card Image.
  6. Click Save & Publish. 

7. Subscriber Section

** Important Note **
 * For this section to be enable Newsletter plugin must be activated.
